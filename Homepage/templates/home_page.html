    The title of my thesis is "A System to Generate Rhythms for Songs Automatically in Rhythm Game". Because in existing Rhythm games, every time players play, the tempo hints(the timings player needs to tap on the screen) will be the same. In order to provide the refreshment in rhythm game, we downloaded 24836 songs from website and extracted rhythm patterns from them using longest common subsequence algorithm in Python programming language.

    Then because we set three different difficulty levels in rhythm game, we used methods to calculate the difficulty scores of each pattern. After calculation, we used different patterns with different difficulty scores in different difficulty levels in rhythm game. Then when players play our rhythm game, we will generate different tempo hints for the song. By the way, we implemented rhythm game on Android system smartphone.
